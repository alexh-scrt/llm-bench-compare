# llm-bench-compare
A lightweight web application that aggregates and visualizes up-to-date benchmark scores (MMLU, HumanEval, MATH, GSM8K, and more) for major open-source LLMs including DeepSeek V3/V4, Kimi, Llama, Mistral, and Qwen variants.
